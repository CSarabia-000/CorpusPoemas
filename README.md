# Creación de un corpus de poemas

En este proyecto se realiza la extracción automática de poemas desde la web para su posterior limpieza y análisis textual.

## Objetivo
Estudiar y poner en práctica técnicas de procesamiento de datos textuales aplicadas a un corpus de poemas.

## Tecnologías
- Python  
- Pandas  
- re  
- nltk  
- spacy  
- BeautifulSoup  
- requests  
- string  
- urllib  
- Matplotlib  
- seaborn  
- Google Colab  

## Estructura
- `Web_scraping.ipynb`: notebook para la extracción de poemas desde la web.  
- `Analisis_estadistico.ipynb`: notebook para el análisis estadístico del corpus.

## Uso
1. Clona el repositorio.
2. Abre el archivo `Analisis_estadistico.ipynb` en Google Colab o Jupyter.
3. Ejecuta las celdas en orden.
